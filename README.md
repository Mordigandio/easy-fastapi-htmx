# Easy FastAPI + HTMX
* Иллюстрационное веб-приложение управления библиотечным каталогом, **разработанное в рамках проектной деятельности** для Института.

## Технологии
* **Бэкенд:** Python, FastAPI, Pydantic v2, Alembic
* **База данных:** PostgreSQL, SQLAlchemy
* **Фронтенд:** HTMX, Jinja2

## Запуск проекта

### 1. Клонирование репозитория
```bash
git clone project_link
```

### 2. Настройка окружения
Создайте файл `.env` на основе примера .env.example

### 3. Установка зависимостей
Для управления пакетами используется `uv`:
```bash
uv sync
```

### 4. Запуск приложения
```bash
python -m uvicorn backend.app.main:app
```
